<h4>ABOUT</h4>
    <p>This projects is a modern backend project built with laravel (php) which helps user submit/raise a ticket concerning a particular. it best appropriate for customer service division or any startup which needs a system to handle user complaints
    </p>

<h4>FEATURES</h4>
    <ol>
        <li>Auth system</li>
        <li>User gets Email Notifiction for Ticket created and closed.</li>
        <li>File attachement (sreenshot or image )</li>
        <li>Admin responds and provides solution, close ticket.</li>
        <li>Users can view their open opened ticket(s)</li>

    </ol>
